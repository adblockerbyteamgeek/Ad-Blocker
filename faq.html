Frequently Asked Questions

Answers to common questions about Ad Block & Boost Pro

🪟 Is this a browser extension?

No. This is a standalone `.exe` application for Windows. It blocks ads system-wide, not just in your browser.

🔒 How does it block ads?

It modifies your system's hosts file to block known ad-serving domains before they can load.

💻 What are the system requirements?

Any modern Windows PC (Windows 7, 10, 11). Must be run as Administrator.

🔑 Can I use one license on multiple PCs?

Each license is intended for a single user on up to 2 personal devices.

🌐 Do I need internet access to use it?

Only for downloading and activation. After that, the app works completely offline.

📦 Are there automatic updates?

Not yet — but we’ll announce updates via email and on this website. You’ll need to re-download the latest version manually.
